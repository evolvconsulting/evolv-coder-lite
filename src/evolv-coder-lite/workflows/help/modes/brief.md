**eCL — top commands**

```text
/ecl:new-project           Initialize a project (greenfield)
/ecl:map-codebase          Map an existing codebase (brownfield)
/ecl:plan-phase <N>        Create a phase plan
/ecl:execute-phase <N>     Execute a phase
/ecl:progress              Where am I, what's next
/ecl:quick                 Small ad-hoc task with eCL guarantees
/ecl:fast "<task>"         Trivial inline task — no subagents
/ecl:debug "<symptom>"     Persistent debug session (survives /clear)
/ecl:capture               Save an idea / todo / note
/ecl:ship <N>              Open a PR from a completed phase
```

More: `/ecl:help` (default tour) · `/ecl:help --full` (everything) · `/ecl:help <topic>` (one section)
